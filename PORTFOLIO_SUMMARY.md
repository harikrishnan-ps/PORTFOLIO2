# Harikrishnan P S - AI/ML Developer Portfolio

## Overview
A professional, modern portfolio website showcasing AI/ML development expertise, projects, and experience.

## Features Implemented

### 1. Navigation
- Fixed navigation bar with smooth scrolling
- Mobile-responsive menu
- Gradient text logo effect

### 2. Hero Section
- Eye-catching introduction with gradient effects
- Call-to-action buttons
- Animated scroll indicator

### 3. About Section
- Three highlight cards showcasing key expertise areas
- Detailed professional summary
- Hover effects on cards

### 4. Education Section
- B.Tech in Computer Science Engineering details
- Institution and timeline information
- Visual icon representation

### 5. Projects Section
- Two featured projects with comprehensive details
- GitHub links for each project
- Technology tags
- Key highlights for each project

### 6. Experience Section
- Timeline-style layout
- Three professional experiences
- Detailed responsibilities for each role

### 7. Skills Section
- Four categorized skill groups:
  - Programming Languages

### 8. Achievements Section
- Three key achievements displayed in cards

### 9. Contact Section

### 10. Footer
- Copyright information
- Technology stack credits

## Design System

### Color Palette
- **Background**: Deep dark blue (#0a0c10 equivalent in HSL)
- **Primary/Accent**: Vibrant blue (#007bff)
- **Text**: Light gray for optimal readability
- **Cards**: Slightly lighter than background for depth

### Typography
- System fonts with fallbacks
- Responsive text sizing (mobile to desktop)
- Clear hierarchy with font weights

### Visual Effects
- Gradient text for headings
- Hover effects with blue glow
- Smooth transitions
- Card shadows and borders

### Responsive Design
- Mobile-first approach
- Breakpoint at xl (1280px) for desktop layouts
- Flexible grid systems
- Collapsible mobile navigation

## Technical Stack
- **Framework**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom design tokens
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Routing**: React Router v7
- **Build Tool**: Vite
