\n## 3. Design Style

### 3.1 Color Scheme
- Primary Background: Deep dark (#0a0c10) for modern tech aesthetic
- Accent Color: Vibrant blue (#007bff) for interactive elements and highlights
- Text Color: Light gray (#f0f0f0) for optimal readability on dark background

### 3.2 Typography
- Primary Font: Inter for body text and headings, ensuring clean readability
- Code/Technical Font: JetBrains Mono for displaying technical skills and code-related content
- Alternative: Space Grotesk for section titles to add visual hierarchy

### 3.3 Visual Details
- Card-based layout for projects and experience sections with subtle shadows
- Rounded corners (8-12px) for modern, approachable feel
- Smooth hover transitions on interactive elements (links, cards)\n- Icon integration for skills and contact methods
- Clean grid layout for skills section with tag-style presentation

### 3.4 Layout Structure
- Single-page scrollable design with fixed navigation menu
- Hero section with name and profile summary at top
- Alternating content blocks for visual rhythm
- Generous whitespace for content breathing room and focus
